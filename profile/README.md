## 先确认：你的 IP 是真的被封了吗？

遇到连不上的情况，别急着换 IP 或者花冤枉钱，先排查一下问题根源。

**方法一：用 ping.pe 做多地 ping 测试**

把你的 VPS IP 输入 ping.pe，它会从全球几十个节点同时 ping 你的服务器。如果海外节点都能 ping 通，只有国内节点全部丢包——恭喜你，大概率就是 IP 被封了。

**方法二：搬瓦工 KiwiVM 官方检测工具**

登录搬瓦工后台，进入 KiwiVM 控制面板，找到你的 VPS-ID（一串数字），然后访问下面这个地址检测：


https://kiwivm.64clouds.com/VPS-ID/main-exec.php?mode=blacklistcheck


把 `VPS-ID` 替换成你自己的 ID。这是搬瓦工官方的 IP 黑名单检测工具，结果比较准确。

**方法三：检查是否是端口被封而不是 IP 被封**

有时候并不是整个 IP 被封，而是某个端口被封。可以用 port.ping.pe 分别测试常用端口（22、443 等）。如果只有特定端口不通，问题就不在 IP 层面，而是端口配置问题，换个端口就能解决。
